# Contributing to NGTube

Thank you for your interest in contributing to **NGTube**.

NGTube is an educational YouTube scraping library focused on **data extraction**, not content downloading. Contributions are welcome as long as they align with the project’s scope and goals.

---

## ⚠️ Important Notes

- This project interacts with **YouTube’s internal APIs**
- YouTube frequently changes its internal structures
- Not all breakages can be fixed immediately
- Contributions must **not** attempt to bypass paywalls, authentication, or private content

By contributing, you acknowledge that this project is for **educational and research purposes only**.

---

## How to Contribute

### 1. Reporting Bugs

Before opening an issue:
- Make sure you are using the **latest version**
- Test with a different video/channel if possible
- Check existing issues for duplicates

When opening a bug report, please include:
- A **YouTube URL** that reproduces the issue
- Your **Python version**
- Your **operating system**
- The **country/localization** used (if any)
- Full error output or traceback

Incomplete issues may be closed without response.

---

### 2. Feature Requests

Feature requests are welcome if they:
- Fit the scope of **data extraction**
- Do not involve downloading copyrighted media
- Do not rely on private or authenticated endpoints

Please describe:
- The problem you are trying to solve
- Why existing functionality is insufficient
- Example output (if applicable)

---

### 3. Pull Requests

Pull requests should:
- Be focused on **one change** (bug fix or feature)
- Follow the existing code structure and style
- Avoid unnecessary refactoring
- Not introduce new heavy dependencies without discussion

Before submitting a PR:
- Ensure the code runs without errors
- Test against multiple videos/channels if relevant
- Keep changes minimal and readable

---

## Code Style Guidelines

- Use **clear, explicit variable names**
- Prefer **readability over cleverness**
- Avoid excessive abstraction
- Keep logic close to where it is used
- Do not add comments that restate obvious code

---

## Legal & Ethical Considerations

Do **not** submit code that:
- Circumvents authentication, logins, or paywalls
- Accesses private or unlisted content
- Attempts to evade rate limits or detection mechanisms
- Violates applicable laws or terms of service

Such contributions will be rejected immediately.

---

## Questions

If you are unsure whether a change is appropriate:
- Open an issue first
- Clearly describe your idea before implementing it

---

Thank you for helping improve NGTube.
